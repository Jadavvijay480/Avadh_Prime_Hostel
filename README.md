# 📌 AVADH_PRIME_Hostel

A complete **Hostel Management System** designed to manage hostel operations like student records, room allocation, fees, attendance, and complaints efficiently.

---

## 🚀 Features

- 👨‍🎓 Student Management  
- 🏨 Room Allocation  
- 💰 Fee Management System  
- 📅 Attendance Tracking  
- 🛠 Complaint System  
- 👮 Warden Management  
- 📊 Admin Dashboard  

---

## 🛠 Technologies Used

- Frontend: HTML, CSS, JavaScript, Bootstrap  
- Backend: ASP.NET Core / MVC / Web API  
- Database: SQL Server  
- Version Control: Git & GitHub  

---

## 📂 Project Structure

AVADH_PRIME_Hostel/
│── Controllers/  
│── Models/  
│── Views/  
│── wwwroot/  
│── Database/  
│── appsettings.json  
│── Program.cs  

---

## ⚙️ Setup Instructions

1. Clone Repository  
   git clone https://github.com/your-username/AVADH_PRIME_Hostel.git  

2. Open in Visual Studio  

3. Update Database Connection in appsettings.json  

4. Run Project (Ctrl + F5)
